Each action consists of:

  • Action Name: A descriptive name (e.g., "Forward Movement")
  • Action Type: Button (discrete) or Joystick (continuous with X, Y values)
  • Tuple List: Multiple (topic, message) pairs that will be published when triggered

Format: One tuple per line: topic_name, message_to_publish

For Joystick Actions: Use X and Y as placeholders for joystick position
